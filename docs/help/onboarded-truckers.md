# Onboarded Truckers — Help

Draft v1. Edit freely.

## What this page is for

This is your **active carrier list** — every trucker who has finished onboarding and is ready to be assigned to loads. Think of it as the difference between "leads I'm working on" (Truckers page) and "carriers I can dispatch today" (this page).

Use it when you're picking who to assign a new load to or who to call about an available lane.

## How to use it — common workflows

### Find an onboarded trucker for a load

1. Search by MC# or legal name in the top search box.
2. Or filter by sales rep using the dropdown to see only one rep's onboarded carriers.
3. Click a row to open the trucker's profile.
4. Cross-check their **Operating States**, **Avoid States**, and **Preferred Lanes** against the load you want to assign.

### Re-assign an onboarded trucker to a different rep / dispatcher

1. Open the trucker.
2. Change the **Sales Agent** or **Dispatcher** dropdown.
3. Click **Save Assignments**.

The new rep / dispatcher gets a notification.

### Update contact info or routes for an active carrier

1. Open the trucker.
2. Edit the profile fields (phone, email, address, preferred lanes, etc.).
3. Save.

## What each button does

This page is the same Truckers list under the hood, just pre-filtered to fully-onboarded carriers — so every button works the same as the Truckers page. See the **Truckers** help for the full button reference.

## When to use this page vs the Truckers page

- Use **Truckers** when you're working leads — calling, sending texts, marking interested.
- Use **Onboarded Truckers** when you have a load and need to pick a carrier who's already documented and ready to drive.
